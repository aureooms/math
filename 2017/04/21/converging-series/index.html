Converging series

Fri, Apr 21, 2017

Theorem

Let \(-1 < x < 1\),

\[ 1 + x + x^2 + x^3 + x^4 + \cdots = \frac{1}{1-x}. \]

Proof

\[ \begin{aligned} (1 - x)(1 + x + x^2 + x^3 + x^4 + \cdots + x^k) &= 1 - x^{k+1}, \\ \lim_{k \to \infty} (1 - x)(1 + x + x^2 + x^3 + x^4 + \cdots + x^k) &= \lim_{k \to \infty} 1 - x^{k+1}, \\ (1 - x)(1 + x + x^2 + x^3 + x^4 + \cdots) &= 1. \end{aligned} \]
